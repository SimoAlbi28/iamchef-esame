# Avvio Progetto IamChef

## Prerequisiti
- Node.js installato
- Java 17+ installato

## Comando per avviare il progetto

Aprire il terminale nella cartella del progetto e lanciare:

```bash
cd frontend && npm install && npm run build && cp -r dist/* ../backend/src/main/resources/static/ && cd ../backend && ./mvnw spring-boot:run
```

## Dove vedere il progetto

- http://localhost:8080
